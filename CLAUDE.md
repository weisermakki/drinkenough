# DrinkEnough – Claude-Anweisungen

## Pflichtverhalten nach jeder Änderung

Nach **jeder** Dateiänderung (Edit, Write) musst du folgende Schritte ausführen, ohne gesondert gefragt zu werden:

1. Alle geänderten Dateien stagen: `git add <dateien>`
2. Commit erstellen mit aussagekräftiger Nachricht
3. Direkt zu GitHub pushen: `git push`

Commit-Nachricht immer auf Deutsch, kurz und präzise.
Co-Authored-By-Zeile anhängen:
`Co-Authored-By: Claude Sonnet 4.6 <noreply@anthropic.com>`

## Sprache

Alle Antworten auf **Deutsch**.

## Projekt-Stack

- React Native mit Expo
- Supabase als Backend
- TypeScript

## Code-Stil

- Keine unnötigen Kommentare
- Keine zusätzlichen Abstraktionen über den Task hinaus
- Keine Emojis im Code außer explizit gewünscht
